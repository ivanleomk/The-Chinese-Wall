# the-chinese-wall

## Getting Started

```bash
# Create a new virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Setup environment variables
cp .env.example .env

# Start the application
python3 main.py
```
